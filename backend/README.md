# Backend

Lambda functions for Neighborly API.

## Technology Stack

- Runtime: Java 17
- Build: Gradle
- Deployment: AWS Lambda

## Structure

```
backend/
├── message-handler/     # Messaging Lambda functions
├── building-handler/    # Building management functions
└── auth-handler/        # Authentication functions
```
